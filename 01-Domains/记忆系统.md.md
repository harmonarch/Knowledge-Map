# 记忆系统.md

## 这个问题域解决什么问题
在有限的 Token 预算内，把最相关的上下文喂给 LLM

## 为什么它重要
因为大模型本身是无状态的，每一次调用都是一张白纸。但 Agent 不是一个一次性的工具，用户会期待他能够记住之前的对话等等。

## 核心子问题
	Agent 的记忆系统 = 分层的存储 + 混合的检索 + 智能的调度 + 异步的写入。
  
	  ▎ 存什么（类型）、怎么找（检索）、什么时候放进去（调度）、什么时候写回去（时机），这四个问题回答清楚了，记忆系统就立住了。
